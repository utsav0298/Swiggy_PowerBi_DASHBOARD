# Swiggy Data Analysis – Power BI Project

## 📊 Overview
This Power BI project provides an end-to-end analysis of Swiggy’s food delivery data to uncover sales trends, restaurant performance, and customer ordering patterns. It helps identify key business insights using interactive visualizations and KPIs.

## 🎯 Objectives
- Analyze order volume and revenue by city, cuisine, and time period  
- Compare restaurant performance metrics  
- Identify customer preferences and delivery trends  
- Build KPIs to track sales growth and profitability  

## ⚙️ Tools & Techniques
- **Power BI Desktop**
- **DAX** for calculated measures and KPIs
- **Excel / CSV** for data preparation and cleaning
- **Data Modeling** for relationships between tables

## 📈 Dashboard Highlights
- KPIs: Total Orders, Total Sales, Avg Order Value  
- Category-wise revenue and order trends  
- City and cuisine-wise insights  
- Monthly sales trend visualization  
- Dynamic filters for period, city, and cuisine
